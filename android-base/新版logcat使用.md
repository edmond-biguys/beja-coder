logcat 使用指令


package:com.xinzailingtech.daggersample level:INFO message:user

package 过滤包名

tag 过滤日志tag

level 过滤日志级别 error info warning error

message 过滤日志内容


同一个类型可以过滤多个

比如同时过滤两个包的日志，或者通过过滤两种message的消息
